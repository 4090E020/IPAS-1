# 社交工程攻擊
```
在電腦科學，社交工程指的是通過與他人的合法交流，來使其心理受到影響，做出某些動作或者是透露一些)【機密資訊】的方式。

這通常被認為是)【欺詐】他人以收集資訊、行騙和入侵電腦系統的行為。

在英美普通法系，這一行為一般是被認作)【侵犯隱私權】的。

歷史上，社交工程是隸屬於)【社會學】，不過其影響他人心理的效果引起了電腦安全專家的注意。
```
